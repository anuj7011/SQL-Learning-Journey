# SELECT Statement

## 📖 Description
The SELECT statement is used to retrieve data from one or more columns of a table.

## 📚 Concepts Covered
- Select all columns
- Select specific columns
- Retrieve required data

## 📝 Syntax

SELECT column_name
FROM employees;

## 💻 Example Query

SELECT name, salary
FROM employees;

## 📊 Expected Output

| name | salary |
|------|--------|
| Anuj | 45000 |
| Ravi | 52000 |

## 🎯 Learning Outcome
Learned how to retrieve data from a database table using the SELECT statement.
