# WHERE Clause

## 📖 Description
The WHERE clause is used to filter records based on a condition.

## 📚 Concepts Covered
- Filter rows
- Comparison operators
- Multiple conditions

## 📝 Syntax

SELECT *
FROM employees
WHERE salary > 50000;

## 💻 Example Query

SELECT *
FROM employees
WHERE city = 'Delhi';

## 📊 Expected Output

| emp_id | name | city |
|--------|------|------|
| 1 | Anuj | Delhi |
| 5 | Mohit | Delhi |

## 🎯 Learning Outcome
Learned how to filter data using the WHERE clause.
