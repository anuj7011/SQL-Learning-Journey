# ORDER BY Clause

## 📖 Description
Used to sort query results.

## 📚 Concepts Covered
- ASC
- DESC

## 📝 Syntax

SELECT *
FROM employees
ORDER BY salary DESC;

## 💻 Example Query

SELECT name,salary
FROM employees
ORDER BY salary ASC;

## 📊 Expected Output

| name | salary |
|------|--------|
| Mohit | 30000 |
| Anuj | 45000 |

## 🎯 Learning Outcome
Learned how to sort data in ascending and descending order.
