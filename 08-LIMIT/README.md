# LIMIT Clause

## 📖 Description
Used to restrict the number of rows returned.

## 📚 Concepts Covered
- Top records
- Row limiting

## 📝 Syntax

SELECT *
FROM employees
LIMIT 5;

## 💻 Example Query

SELECT *
FROM employees
LIMIT 3;

## 📊 Expected Output

| emp_id | name |
|--------|------|
| 1 | Anuj |
| 2 | Ravi |
| 3 | Mohit |

## 🎯 Learning Outcome
Learned how to limit query results.
